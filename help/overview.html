Overview

Database(Postgres ) based tool for processing hsrr readings.

  • Fits readings to network.
  • Splits requested sections of network into 100m pieces and finds average values
  • Output is in resized table

How to Use

Under 'setup':

  • Connect to a database
  • Setup the database if this has not already been done.
  • The network is in table network. Import this into qgis then set network layer to this.
  • Import the readings table into qgis then set readings layer to this.

Under 'upload':

  • Upload the readings

Under 'fitting':

  • Fit readings to network.(see fitting )

Under 'lengths and missing':

  • Check missing sections.
