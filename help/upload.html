Uploading readings

The upload readings tab is for uploading readings and editing the run_info table.

Readings can be uploaded from spreadsheets. "upload folder" will try to upload every .xsl in a folder and subfolders.

The new run will be given a unique name based on filename. The plugin will try to name example.xls as example then example_1,example_2...

The readings will be added to the readings table.

A new row will be added to the run_info table.
