Setup tab

Connect to a database through the "connect..." dialog of the setup tab

The "Prepare New Database..." dialog is to set up a database for the first time


This plugin can select sections on a layer with the road network.To do this:

  • Import the network table into QGIS.
  • Set "Network layer" to this layer
  • Set "section label" to "sec"


This plugin can select readings on a layer. To do this:

  • Import the readings table into QGIS.
  • Set "readings Layer" to this layer
  • Set "Run field" to "run"
  • Set "f_line field" to "f_line"
